# 기능 명세

## 1. 오늘의 운세 자동 조회

**동작 정의**
- 사용자가 최초 1회 생년월일(양력/음력)을 입력하면 저장
- 매일 아침 오하아사 or 사주 데이터 기반으로 오늘의 운세 자동 조회
- 운세에서 오늘의 행운색 추출

**출력값**
- 오늘의 한줄 운세
- 오늘의 행운색 (예: 파란색, 붉은 계열 등)

---

## 2. 날씨 자동 조회

**동작 정의**
- 최초 1회 위치 권한 허용 or 지역 수동 입력
- 매일 아침 현재 위치 기반 날씨 API 호출
- 기온, 날씨 상태(맑음/흐림/비/눈) 조회

**출력값**
- 현재 기온 + 최고/최저 기온
- 날씨 상태 (아이콘 + 텍스트)
- 옷차림 힌트 (예: "겉옷 필요", "우산 챙기세요")

---

## 3. 행운색 기반 옷 추천

**동작 정의**
- 운세에서 추출한 행운색을 기준으로 내 옷장에서 해당 색상 옷 필터링
- 날씨 조건(기온, 강수)에 맞는 옷 종류(코트/셔츠/반팔 등) 추가 필터링
- 두 조건을 모두 만족하는 옷 우선 추천, 없으면 조건 완화하여 추천

**출력값**
- 추천 상의 1~3개
- 추천 하의 1~3개
- 추천 아우터 (기온 기준)
- 행운색 포인트 설명 (예: "오늘 파란색이 행운을 가져다줘요")

---

## 4. 내 옷장 등록

**동작 정의**
- 직접 이미지 업로드 (카메라 촬영 or 갤러리)
- 쇼핑몰 상품 이미지 URL 붙여넣기로 저장
- 옷 등록 시 색상 태그 자동 감지 (수동 수정 가능)
- 카테고리 분류: 상의 / 하의 / 아우터 / 신발 / 액세서리

**출력값**
- 등록된 옷 목록 (그리드 뷰)
- 색상별 / 카테고리별 필터
