# 기능 우선순위

## P0 — 없으면 서비스 안 됨

1. **날씨 조회** — 기온/날씨 상태 표시
2. **운세 조회 + 행운색 추출** — 오하아사 연동
3. **옷장 등록** — 이미지 업로드 + 색상 수동 태그

## P1 — MVP 핵심

4. **행운색 + 날씨 기반 추천 로직** — 두 조건 합산해서 옷 필터링
5. **초기 설정 흐름** — 생년월일 입력, 위치 설정

## P2 — 있으면 좋음

6. **쇼핑몰 URL로 옷 저장**
7. **색상 자동 감지** (AI 이미지 분석)
8. **아침 알림 (푸시)**

## P3 — 나중에

9. **멀티유저 지원**
10. **사주 기반 운세**
11. **코디 히스토리** (오늘 뭐 입었는지 기록)
