# 사용자 시나리오

## 기본 시나리오: 아침에 오늘의 옷 고르기

**Who**: 아침마다 옷 고르는 게 번거롭고, 운세와 행운색을 참고해서 옷을 고르는 사람

**When**: 매일 아침 출근/외출 준비 시간 (7~9시)

**Where**: 스마트폰 or 노트북 웹브라우저

**Why**: 날씨와 운세를 따로 확인하고 옷을 고르는 게 번거로워서

**How**:
1. 웹사이트 접속
2. 메인 화면에서 오늘의 날씨 + 운세 + 행운색 한눈에 확인
3. 내 옷장에서 행운색 + 날씨에 맞는 추천 옷 확인
4. 추천 옷 중 마음에 드는 것 선택 → 완료

---

## 초기 설정 시나리오: 처음 사용할 때

1. 생년월일 입력 (운세 계산용)
2. 위치 권한 허용 or 지역 입력 (날씨 조회용)
3. 옷장 등록 (이미지 업로드 or 쇼핑몰 URL)
4. 설정 완료 → 메인 화면으로

---

## 옷장 추가 시나리오: 새 옷 샀을 때

1. 쇼핑몰에서 구매한 옷 상품 페이지 이미지 URL 복사
2. "옷장에 추가" → URL 붙여넣기
3. 색상 태그 확인 (자동 감지) → 필요시 수정
4. 카테고리 선택 (상의/하의/아우터 등)
5. 저장 완료
