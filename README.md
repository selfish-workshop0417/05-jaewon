# 오늘 뭐 입지?

운세와 날씨를 기반으로 오늘의 옷을 추천해주는 개인 웹서비스

## 프로젝트 정보
- **유형**: 개인 도구
- **타겟 사용자**: 나 (개인용)
- **출시 목표**: 미정

## 핵심 기능
- 오늘의 운세 자동 조회 (오하아사 or 사주 기반)
- 날씨 자동 조회 (위치 기반)
- 행운색 기반 옷 추천
- 내 옷장 등록 + 쇼핑몰 이미지 저장 기능

## 폴더 구조 (5개 팀)
- `planning/` — 기획 팀: 무엇을 만들 것인가
- `design/` — 디자인 팀: 어떻게 보여줄 것인가
- `cx/` — CX 팀: 누가 어떻게 사용할 것인가
- `marketing/` — 마케팅 팀: 어떻게 알릴 것인가
- `strategy/` — 전략 팀: 어떻게 지속 가능하게 만들 것인가
